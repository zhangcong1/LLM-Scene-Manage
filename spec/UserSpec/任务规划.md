# 任务规划

## 1. 生成界面UI风格规范文档
使用 ui-ux-pro-max skill 生成界面UI风格规范文档，包括颜色token，间距规则，字号层级，布局栅格，组件规范，交互规范等元素的标准化指导。将生成后的设计方案写入到 UI设计风格.md 文件中。整体设计风格需要体现色彩和人工智能元素。前端框架使用 React + Tailwind CSS
## 四、 界面布局风格
### 1. 视觉风格
 * AI 元素： 体现色彩和人工智能的元素。
 * 主题切换： 支持白天（Light）/ 暗夜（Dark）模式切换。
 * 整体布局： 界面符合门户网站风格，要包含以下入口，每个入口是一个单独的tab页面
 1、首页：首页有该系统的介绍，大模型应用场景检索、大模型应用场景创建的入口、热门场景（点击率较高的场景）
 2、发现：对应大模型应用场景检索与查看
 3、我的场景：用户自己创建的应用场景，可再次编辑（固化申请前）
 4、收藏：用户收藏的应用场景；
 5、创建场景：创建场景图标要明显
 6、用户头像：下面有个人中心、消息通知等操作页面；
